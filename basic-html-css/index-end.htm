Upcoming Classes

  • Intro to Relational Databases

    With our Introduction to Relational Databases class, you'll learn the foundation of relational databases so that you can design, create, and store information, and manipulate that information into knowledge. We'll start out with modeling and design, using Entity Relationship Diagrams. Then, you'll learn ANSI SQL to create your database, store your data, change your data, and ask questions of your data. Although we'll focus on MySQL and MySQL Workbench (an open source database, and one of the top databases used in the world), most of the skills and techniques learned will be universal (and if they're not, we'll try and point out what differences there might be with other popular databases). We focus solely on the database end, so how to hook everything up to your application will be out of scope; but we should give you almost everything else you need to get there.

  • Intermediate HTML and CSS

    Through the course of this 4 session class, you'll get a thorough introduction to HTML5 and CSS3 to build on your previous knowledge. You'll get hands-on experience working with HTML5 features like semantic elements and forms, and you'll learn new CSS3 properties and abilities including transformations, transitions, and animation. Finally, we'll dig into responsive design and browser compatibility.
